1 MISSING VALUES

MCAR Completely Random
No pattern. Coffee-spill.
→ Drop if <5% · Mean/median OK
MAR Depends on Other Cols
Group X has more NaN than Y.
→ Group-specific median · KNN impute
MNAR Depends on Itself
High earners skip salary field.
→ Flag was_missing + fill · Flag IS a feature
···

2 DISTRIBUTIONS

SKEW |skew| > 1
Linear models break. Trees immune.
→ np.log1p() for linear · Skip for trees
IMBALANCE Target 84/16%
Accuracy is a TRAP. Model ignores minority.
→ class_weight='balanced' · F1/AUC not accuracy
UNEXPECTED age=-3, $0 salary
Not outliers — ERRORS. Treat as NaN.
→ Replace with NaN · Then impute
···
Linear: needs symmetric, scaled
Trees: handles skew + outliers natively

3 OUTLIERS

REMOVE Only if impossible
"Age can't be negative" ✓ · "Seems high" ✗
WINSORIZE Cap at 1st/99th pct
Keeps rows · Limits influence · Good for linear
LOG Compress right-skewed
log1p() · Ratios > absolutes · Reversible
KEEP If it's the signal
Fraud IS outliers · Use trees or RobustScaler
···
Ask: impossible → real → is it the target?
Linear? Fix it. Trees? Leave it.
